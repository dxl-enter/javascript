1. 如下三段代码能正常执行吗？
```
function dxl(){
  dxl();
}
dxl();
function dxl2(){
  setTimeout(dxl2);
}
dxl2();
function dxl3(){
  Promise.resolve().then(dxl3);
}
dxl3();
```
* 解析
```
1. dxl会堆栈溢出。因为整体代码都在同步执行栈中，造成新生代空间分配资源被全部耗尽。
2. dxl2可以正常执行。因为dxl2实际上被移动到了macrotask中，并未影响到主渲染线程。
3. dxl3可以运行但是造成页面的卡顿。因为当前在这一轮任务里不停地创建微任务，执行，创建，执行，创建.......虽然不会爆栈，但也无法去执行下一个任务，主线程被卡，所以页面会严重卡顿。
```

3. 浏览器的组成
   * shell（用户能操作的部分）
   * 内核（用户看不到的，能处理代码）
     * 渲染引擎（语法规则和渲染）：从上到下一层一层的绘制，浏览器帧频16毫秒切换一次，主要负责html、css基础语法的识别以及绘制
     * js引擎：2001年发布ie6，首次实现对js引擎的优化和分离（代码提升到万行以上）；
     2008年google发布最新浏览器chrome，他是采用优化后的js引擎(代号V8)（s速度很快，直接把js代码转化成0101码）；
